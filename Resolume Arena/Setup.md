# Resolume Arena and Art-Net Integration

## Overview

Resolume Arena was integrated with the WLED lighting system using the Art-Net protocol, enabling real-time control of addressable LED strips directly from visual content generated within Resolume.

This integration allowed the LED installation to function as an external lighting output, extending visual effects beyond the computer display and into the physical environment.

## Motivation

The primary goal of this integration was to explore how professional visual content software could interact with a DIY lighting system.

By combining Resolume Arena and WLED through Art-Net, visual effects created inside Resolume could be transmitted over the network and displayed on the LED strips in real time.

This expanded the system from a simple ambient lighting setup into a network-controlled visual display platform.

## Art-Net Overview

Art-Net is a lighting control protocol that transmits DMX data over standard IP networks.

It is widely used to control lighting devices and pixel-based installations from software applications.

In this project, Art-Net packets generated by Resolume Arena were received by WLED and translated into LED color data for the connected strips.

## Integration Process

### WLED Configuration

WLED was configured to accept Art-Net network data.

The following settings were configured:

* LED Count
* Segment Configuration
* Network Connectivity
* Art-Net Reception

Once configured, WLED became capable of receiving lighting data directly from external software.

### Resolume Configuration

Resolume Arena was configured to output Art-Net data over the local network.

The setup process included:

* Creating output mappings
* Configuring Art-Net transmission
* Assigning universes
* Mapping pixels to LED positions

After configuration, visual effects generated in Resolume were transmitted directly to the LED installation.

## LED Mapping

The LED strips were treated as an output surface for Resolume content.

By mapping visual content to LED positions, effects displayed on the software output could be reproduced on the physical lighting installation.

Examples included:

* Color gradients
* Motion effects
* Wave animations
* Pulse effects
* Dynamic visual patterns

## Network Communication

All communication occurred through the local Wi-Fi network.

The WLED device and the computer running Resolume were connected to the same network, allowing Art-Net packets to be transmitted without additional hardware interfaces.

## Challenges Encountered

Several configuration challenges were encountered during testing and integration.

### Device Discovery

Initial testing required verification that Art-Net packets were being transmitted and received correctly across the network.

### Output Mapping

Accurate LED mapping was necessary to ensure visual effects aligned correctly with the physical LED layout.

### Segment Configuration

Proper segmentation within WLED improved flexibility and enabled more precise control of different LED regions.

### Network Troubleshooting

Multiple rounds of testing were performed to verify:

* Device communication
* Art-Net transmission
* Network visibility
* Data reception

These steps were necessary to achieve reliable real-time operation.

## Applications

The Art-Net integration enabled the LED system to respond directly to visual content rather than relying solely on audio or predefined effects.

Potential applications include:

* Visual experimentation
* Creative lighting effects
* Interactive room ambience
* Real-time content visualization
* Software-controlled LED animations

## Lessons Learned

* Art-Net provides a flexible method for controlling LEDs over standard networks.
* Proper network configuration is critical for reliable communication.
* LED segmentation improves control and mapping flexibility.
* Visual software can be integrated successfully with hobbyist lighting systems using open protocols.
* Network-based lighting control opens possibilities beyond traditional audio-reactive effects.

## Result

The integration successfully enabled real-time control of the LED strips from Resolume Arena through Art-Net, allowing visual content created in software to be reproduced directly on the physical lighting installation through the local network.
