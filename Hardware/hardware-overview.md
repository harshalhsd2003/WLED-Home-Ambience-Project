# Hardware Overview

## Project Description

This project is a DIY home ambient lighting system built around WLED and ESP32. The setup combines addressable LED strips, audio-reactive effects, screen ambience synchronization, and Art-Net integration to create customizable lighting experiences for music, media consumption, and room ambience.

## Hardware Components

| Component                 | Purpose                                   |
| ------------------------- | ----------------------------------------- |
| ESP32                     | Main controller running WLED              |
| WS2812B LED Strips        | Addressable RGB lighting                  |
| MAX9814 Microphone Module | Audio input for sound-reactive effects    |
| 12V 10A SMPS              | Primary power supply                      |
| Buck Converter            | Voltage conversion from 12V to 5V         |
| Breadboard                | Prototyping and connections               |
| Jumper Wires              | Signal and power connections              |
| Home Wi-Fi Network        | Wireless control and software integration |

## System Architecture

12V 10A SMPS
↓
Buck Converter (12V → 5V)
↓
ESP32 + MAX9814 + WS2812B LED Strips

## Wiring Summary

### Power Distribution

* ESP32 powered from a regulated 5V supply generated by the buck converter.
* MAX9814 microphone module powered from the same 5V supply.
* Common ground shared between ESP32, microphone module, and LED strips.

### LED Connections

* GPIO D4 used as the data output for the first LED strip segment.
* GPIO D16 used as the data output for the second LED strip segment.
* Each data line includes a 220Ω series resistor for signal protection and improved reliability.

### Microphone Connections

* MAX9814 powered from the ESP32 5V and GND rails.
* Microphone signal output connected to the ESP32 audio input pin configured in WLED Sound Reactive firmware.

### Prototyping Method

* All connections implemented using breadboard and jumper wires.
* System designed as a flexible prototyping platform for testing different lighting configurations and software integrations.
